# コントローラ UI

対象読者: 画面に何が表示され、どう操作できるかを確認したい人。

## Why

動画の上に常時ボタン類を出しっぱなしにすると視聴の邪魔になる。かといって完全にUIを排除すると
マウス操作だけの人が操作できなくなる。このアプリは「必要な時だけ出て、使わなければ自動的に消える」
オーバーレイUIと、それと等価なキーボードショートカットの両方を提供することで、この二律背反を解決している。

## 画面に出ているもの

下部に**コントローラ帯**（クリック可能なボタン列）、上部に**ナビゲーション帯**（一覧タブ・ログインボタン）が
動画に重なって表示される。マウスで直接クリック/ドラッグして操作できる（キーボード操作は代替手段であり必須ではない）。

### 下部コントローラ帯

| 表示 | 操作 | 内容 |
|------|------|------|
| ▶ / ⏸ | クリック | 再生・一時停止 |
| シークバー（帯上部・全幅） | クリック/ドラッグ | 任意位置へシーク（シーク不可な配信ではドラッグ無効） |
| `● ライブ` バッジ（ライブ時） | クリック | ライブ最新地点へジャンプ。追いつくと赤、遅れていると白で表示 |
| `mm:ss / mm:ss`（通常再生時） | — | 再生位置と長さの表示のみ |
| 👍 | クリック | 現在の動画に高評価（[login-and-rating.md](login-and-rating.md)） |
| 💬 チャット / 💬 非表示 | クリック | チャットパネルの開閉（利用可能な動画のみ表示。[chat.md](chat.md)） |
| 音量バー | クリック/ドラッグ | 音量調整 |
| 🔊 / 🔇 | クリック | ミュート切替 |
| `コーデック: 自動` 等 | クリック | コーデックを順に切替（[playback.md](playback.md)） |
| `画質: 自動` 等 | クリック | 画質を順に切替（[playback.md](playback.md)） |

### 上部ナビゲーション帯

| 表示 | 操作 | 内容 |
|------|------|------|
| `📋 おすすめ` | クリック | おすすめ一覧を開く（候補がある時のみ表示） |
| `📃 再生リスト` / `📺 登録チャンネル` / `🕘 履歴` | クリック | 各一覧を開く（ログイン時のみ表示。[browse-lists.md](browse-lists.md)） |
| ログインボタン（未ログイン時、右寄せ） | クリック | ログイン開始（[login-and-rating.md](login-and-rating.md)） |
| `A+` / `A-`（チャット表示時） | クリック | チャットの文字サイズ変更（[chat.md](chat.md)） |

一覧オーバーレイを開いているときは、右上の `✕` ボタンで閉じられる（詳細: [browse-lists.md](browse-lists.md)）。

## キーボードショートカット（マウスと同じ操作の代替手段）

| キー | 動作 |
|------|------|
| 英数字 / 記号 | URL 欄へ入力（URL は空白を含まないため Space は再生に使える） |
| `Ctrl`+`V` | クリップボードの URL を貼り付け |
| `Enter` | URL 欄の URL を再生 |
| `Space` | 再生 / 一時停止 |
| `←` / `→` | 5 秒シーク |
| `↑` / `↓` | 音量 ±5 |
| `Tab` | 一覧オーバーレイの開閉 |
| 一覧中 `1`/`2`/`3`/`4` | ソース切替（登録新着 / おすすめ / 履歴 / 再生リスト） |
| 一覧中 `↑`/`↓`・クリック | 項目選択 |
| 一覧中 `Enter` | 選択を再生（再生リストは中身を開く） |
| 一覧中 `Backspace` | 再生リストの中身から一覧へ戻る |
| `Ctrl`+`T` | チャットの開閉 |
| `Ctrl`+`L` | ログイン開始 |
| `Ctrl`+`G` | 現在の動画に高評価 |
| `Ctrl`+`Q` / `Ctrl`+`C` | 画質 / コーデックの切替（再生中なら取り直す） |
| `Ctrl`+`-` / `Ctrl`+`+` | コメント（チャット）の文字サイズを縮小 / 拡大 |
| `Esc` | 一覧を閉じる / URL 欄をクリア |

## 自動非表示

約 3 秒間マウス/キー操作がないと、コントローラ帯・ナビゲーション帯・URL欄を自動的に隠す（マウス移動や
キー入力で再表示）。一覧・チャット表示中は隠さない。動画本体の視界を最大限確保しつつ、必要な時に
すぐ操作できるようにするための挙動。

## 入力の制限

日本語 IME は未対応。URL 欄は ASCII 入力のみを想定している（検索機能自体が未実装のため、実運用上は
URL を貼り付ける運用になる）。
