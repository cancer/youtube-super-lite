# コントローラ・キーボード操作

対象読者: 通常操作のキーバインドと UI の挙動を確認したい人。

## 画面構成

動画は全画面表示。下部のコントローラ帯に再生/一時停止・シークバー・時間・画質/コーデック表示を重ねる。
一覧・チャットは同じオーバーレイ層（子窓 + DirectComposition、[design/overlay-rendering.md](../design/overlay-rendering.md)）
で描画する。チャット表示中は動画の描画領域自体を左に縮め、右側にチャットパネルを表示する（真の左右分割）。

## キーバインド

| キー | 動作 |
|------|------|
| 英数字 / 記号 | URL 欄へ入力（URL は空白を含まないため Space は再生に使える） |
| `Ctrl`+`V` | クリップボードの URL を貼り付け |
| `Enter` | URL 欄の URL を再生 |
| `Space` | 再生 / 一時停止 |
| `←` / `→` | 5 秒シーク |
| `↑` / `↓` | 音量 ±5 |
| `Tab` | 一覧オーバーレイの開閉 |
| 一覧中 `1`/`2`/`3`/`4` | ソース切替（登録新着 / おすすめ / 履歴 / 再生リスト） |
| 一覧中 `↑`/`↓`・クリック | 項目選択 |
| 一覧中 `Enter` | 選択を再生（再生リストは中身を開く） |
| 一覧中 `Backspace` | 再生リストの中身から一覧へ戻る |
| `Ctrl`+`T` | チャットの開閉 |
| `Ctrl`+`L` | ログイン開始 |
| `Ctrl`+`G` | 現在の動画に高評価 |
| `Ctrl`+`Q` / `Ctrl`+`C` | 画質 / コーデックの切替（再生中なら取り直す） |
| `Ctrl`+`-` / `Ctrl`+`+` | コメント（チャット）の文字サイズを縮小 / 拡大 |
| `Esc` | 一覧を閉じる / URL 欄をクリア |

## 自動非表示

約 3 秒間マウス/キー操作がないと、コントローラ帯・URL欄を自動的に隠す（マウス移動やキー入力で再表示）。
一覧・チャット表示中は隠さない。

## 入力の制限

日本語 IME は未対応。URL 欄は ASCII 入力のみを想定している（検索機能自体が未実装のため、実運用上は
URL を貼り付ける運用になる）。
